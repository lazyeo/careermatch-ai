# Product Charter — CareerMatch AI

## 1) Product Positioning

CareerMatch AI是一个基于人工智能的智能简历优化与岗位匹配平台，专注于新西兰市场，为18-45岁的求职者提供从简历创建到岗位申请的全流程求职解决方案。我们不仅提供简历优化服务，更通过岗位收集、智能匹配、申请追踪等功能，成为求职者的全方位职业发展伙伴，显著提升求职成功率和效率。

## 2) Brand Keywords

- **智能匹配** — AI深度分析岗位要求与用户背景，提供精准的匹配度评分和优化建议
- **全流程覆盖** — 从简历创建、岗位发现、申请追踪到面试准备的完整求职生态
- **专业化** — 基于新西兰本土招聘市场特点，提供专业的求职指导和优化建议
- **高效便捷** — 浏览器扩展一键收集岗位，AI自动生成定制简历和求职信
- **数据驱动** — 基于量化匹配分析和成功率数据，为用户提供最优求职策略

## 3) Core Problem / JTBD

当求职者寻找心仪工作时，他们面临四大核心挑战：1) 岗位信息分散在多个平台难以统一管理；2) 不确定自己与岗位的匹配度，导致盲目投递浪费时间；3) 简历缺乏针对性，无法突出与特定岗位的契合点；4) 面试准备盲目，不知道该重点准备什么。传统求职方式效率低下，缺乏科学的匹配评估和个性化指导，导致优秀候选人错失机会，求职周期拉长。

## 4) Goals & Mission

- **Mission:** 让每个人都能找到最匹配的工作机会，用AI赋能职业发展，成为新西兰领先的智能求职平台
- **Desired Outcomes (descriptive):**
  - 求职者能够高效发现和管理心仪岗位，实现岗位信息的统一收集和分析
  - 通过量化匹配分析，用户清楚了解自身竞争力，提高申请的精准性和成功率
  - AI自动生成针对性简历和求职信，从手动2小时缩短到5分钟，大幅提升申请效率
  - 提供个性化面试准备建议，帮助用户在面试中脱颖而出，提升录取概率40%+

## 5) Solutions We Own

### 智能岗位收集与管理
- **What it solves:** 解决岗位信息分散在多个招聘平台，难以统一管理和对比分析的问题
- **Typical path:** 用户通过浏览器扩展一键保存岗位信息，或手动添加岗位URL，系统自动提取结构化信息并进行统一管理
- **Outcome for the user:** 获得统一的岗位管理中心，支持筛选、排序、状态追踪等功能
- **Boundaries:** 专注于新西兰主要招聘网站（Seek、TradeMe、LinkedIn、Indeed），不涉及国际市场
- **Guiding principles:** 数据准确性 / 用户隐私保护 / 平台兼容性

### AI深度岗位分析与匹配
- **What it solves:** 解决求职者无法准确评估自身与岗位匹配度，导致盲目投递的问题
- **Typical path:** AI对岗位进行9维度深度分析，包括角色定位、技能要求、匹配度计算等，生成详细的分析报告和优化建议
- **Outcome for the user:** 获得量化的匹配度评分（0-100）和具体的优势/差距分析，做出明智的申请决策
- **Boundaries:** 基于用户真实简历进行匹配分析，不提供虚假信息或过度乐观的评估
- **Guiding principles:** 客观性 / 可解释性 / 行动导向

### 定制化简历与求职信生成
- **What it solves:** 解决传统简历缺乏针对性，无法突出与特定岗位相关性的问题
- **Typical path:** 基于岗位分析结果，AI自动调整简历结构、优化内容描述、嵌入关键词，同时生成个性化求职信
- **Outcome for the user:** 获得高度匹配的定制简历和专业求职信，显著提升申请通过率
- **Boundaries:** 基于用户真实经历进行优化，不编造虚假信息或过度夸大成就
- **Guiding principles:** 真实性 / 相关性 / ATS友好

### 全流程申请追踪与面试准备
- **What it solves:** 解决求职进度管理混乱和面试准备盲目的问题
- **Typical path:** 系统追踪每个岗位的申请状态，提供时间线管理，并基于岗位特点生成面试问题预测和准备建议
- **Outcome for the user:** 获得完整的申请进度可视化和个性化的面试准备方案，提升面试成功率
- **Boundaries:** 专注于面试准备建议和进度管理，不提供实际的面试代理服务
- **Guiding principles:** 系统性 / 针对性 / 持续改进

### 智能岗位推荐系统
- **What it solves:** 解决求职者被动等待机会，错失潜在合适岗位的问题
- **Typical path:** 基于用户简历和求职偏好，AI主动从岗位库中筛选匹配的新机会，定期推送给用户
- **Outcome for the user:** 获得个性化的岗位推荐，发现更多适合的工作机会，缩短求职周期
- **Boundaries:** 推荐范围限于平台收录的岗位，需要用户主动确认和申请
- **Guiding principles:** 精准性 / 及时性 / 多样性
