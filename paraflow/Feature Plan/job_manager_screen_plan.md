# 岗位管理
提供岗位收集、分析、筛选和状态管理的综合界面，支持多种视图模式和智能筛选功能。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Main Content Area

## Top Navigation Bar
- CareerMatch AI logo
- 主导航菜单：仪表盘、岗位管理、简历管理、个人档案
- 全局搜索框（岗位快速搜索）
- 通知铃铛图标
- 用户头像和下拉菜单

## Page Header Section
- 页面标题："岗位管理"
- 岗位统计概览：
  - 总岗位数量
  - 高匹配度岗位数量（>80%）
  - 待申请岗位数量
  - 本周新增岗位数量
- 添加岗位按钮：支持URL导入和手动创建

## Filter and View Controls
- 筛选控制面板：
  - 申请状态筛选：全部、草稿、已投递、面试中、已录取、已拒绝、已撤回
  - 匹配度范围滑块：0-100分可调节区间
  - 岗位类型多选：软件工程师、产品管理、数据分析等
  - 公司规模筛选：初创公司、中型企业、大型企业
  - 发布时间筛选：今天、本周、本月、更早
  - 薪资范围筛选：可调节薪资区间
- 视图切换按钮：列表视图、卡片视图、看板视图
- 排序选项：最新收集、匹配度高低、申请截止时间、薪资高低

## Job Listings Area
- 岗位卡片网格（卡片视图）：
  - 岗位标题和公司名称
  - 公司logo和位置信息
  - 匹配度评分显示（环形进度条+百分比）
  - 星级评定（1-5星）
  - 薪资范围和福利亮点
  - 发布时间和申请截止日期
  - 申请状态标签（不同颜色区分）
  - 快速操作按钮：查看详情、生成简历、提交申请
- 岗位列表（列表视图）：
  - 紧凑的表格形式显示
  - 批量选择功能
  - 批量操作选项：删除、更改状态、导出
- 看板视图（按申请状态分列）：
  - 草稿、已投递、面试中、已录取、已拒绝、已撤回六列
  - 拖拽功能更新申请状态
  - 每列显示岗位数量

## Bulk Actions Bar
- 批量选择控制：全选、取消全选、反选
- 批量操作按钮：
  - 批量删除选中岗位
  - 批量更改申请状态
  - 批量导出岗位信息
  - 批量生成匹配报告

## Pagination and Load More
- 分页控制器：页码导航和每页显示数量选择
- 加载状态指示器
- 回到顶部浮动按钮
