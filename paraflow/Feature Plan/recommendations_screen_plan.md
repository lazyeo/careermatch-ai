# 智能推荐
基于用户画像和AI算法的个性化岗位推荐页面，提供精准的职业机会发现和推荐理由分析。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Main Content Area

## Top Navigation Bar
- CareerMatch AI logo
- 主导航菜单：仪表盘、岗位管理、简历管理、个人档案
- 全局搜索框
- 通知铃铛图标
- 用户头像和下拉菜单

## Page Header Section
- 页面标题："为您智能推荐"
- 个性化推荐说明：基于您的技能、经验和求职偏好
- 推荐更新时间：最后更新时间和下次更新预期
- 推荐设置按钮：调整推荐偏好和频率

## Recommendation Filters
- 推荐类型筛选：
  - 全部推荐
  - 高匹配度推荐（>85%）
  - 新发布岗位
  - 隐藏机会（非公开招聘）
  - 职业提升机会
- 岗位类型筛选：根据用户历史偏好预设
- 地理位置筛选：新西兰主要城市选择
- 薪资期望筛选：基于用户档案预填
- 工作类型筛选：全职、兼职、合同、远程

## Featured Recommendations
- 今日精选推荐标题
- 顶级推荐岗位卡片：
  - 大型展示卡片格式
  - 岗位标题和公司品牌
  - 超高匹配度标识（>90%）
  - 推荐理由简要说明
  - 薪资亮点和福利优势
  - 申请紧急度提示
  - 立即查看详情按钮

## All Recommendations
- 全部推荐岗位标题
- 推荐岗位卡片网格：
  - 岗位基本信息（标题、公司、地点）
  - 匹配度评分和星级显示
  - 推荐标签（如：技能匹配、经验相符、文化契合）
  - 薪资范围和关键福利
  - 发布时间和热度指标
  - 推荐理由摘要
  - 查看详情和快速申请按钮
- 无限滚动加载更多推荐

## Recommendation Insights
- 推荐算法洞察面板：
  - 您的技能标签云：基于简历分析的核心技能
  - 推荐依据说明：算法如何匹配的透明化解释
  - 市场趋势分析：新西兰就业市场的相关趋势
  - 技能提升建议：基于推荐结果的学习建议

## Recommendation Actions
- 批量操作控制：
  - 批量添加到岗位管理
  - 批量标记为不感兴趣
  - 批量生成匹配分析
- 反馈控制：
  - 推荐质量评价（点赞/点踩）
  - 不感兴趣原因选择
  - 推荐偏好调整建议

## Recommendation Settings Panel
- 推荐频率设置：每日、每周、实时推送
- 推荐类型权重：技能匹配、薪资期望、地理位置、公司规模
- 排除条件设置：不感兴趣的公司和岗位类型
- 通知偏好：邮件、应用内推送、手机短信
