# 简历管理中心
提供用户创建、编辑、管理多版本简历的综合界面，支持AI优化建议和多种模板选择。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Main Content Area

## Top Navigation Bar
- CareerMatch AI logo
- 主导航菜单：仪表盘、岗位管理、简历管理、个人档案
- 全局搜索框
- 通知铃铛图标
- 用户头像和下拉菜单

## Page Header Section
- 页面标题："我的简历"
- 简历统计概览：
  - 简历总数量（包含版本数统计）
  - 基础简历完整度百分比
  - 最近更新时间
  - 使用次数统计（基于生成定制简历次数）
- 快速操作按钮：
  - 新建简历（从头创建）
  - 上传简历（PDF/DOCX文件）
  - 从LinkedIn导入
  - 模板库浏览

## Resume List Section
- 简历卡片列表：
  - 基础简历卡片（Master Resume）：
    - 简历缩略图预览
    - 简历标题和创建时间
    - 完整度进度条（85%等）
    - AI优化建议数量提示
    - 使用统计（基于X个岗位生成）
    - 操作按钮：查看、编辑、导出、复制
  - 版本简历卡片：
    - 版本名称（如"软件工程师版"）
    - 基于关系显示（"基于：基础简历"）
    - 创建和最后修改时间
    - 应用岗位数量统计
    - 匹配度范围（如"平均匹配度 87%"）
    - 操作按钮：查看、编辑、导出、删除

## AI Optimization Panel
- 优化建议卡片：
  - 待优化项目列表：
    - 添加量化成果数据
    - 补充缺失技能标签
    - 优化工作描述用词
    - 调整章节顺序
  - 批量优化按钮
  - 智能检测结果：ATS友好度评分

## Template Gallery Section
- 模板分类选项卡：
  - 按行业（技术、金融、创意、教育、通用）
  - 按风格（传统、现代、创意、ATS优化）
  - 按用途（应届生、转行、高级职位）
- 模板预览网格：
  - 模板缩略图
  - 模板名称和适用说明
  - 用户评分和使用次数
  - 预览和使用按钮

## Recent Activities
- 最近操作时间线：
  - 简历创建和编辑记录
  - AI优化应用历史
  - 简历导出和分享记录
  - 版本创建和删除操作
- 活动筛选：按时间、按类型、按简历版本

## Bulk Operations
- 批量选择控制：选择多个简历版本
- 批量操作选项：
  - 批量导出（打包下载）
  - 批量应用AI优化
  - 批量更改模板
  - 批量删除（非基础简历）
