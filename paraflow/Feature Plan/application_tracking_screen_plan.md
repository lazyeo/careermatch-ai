# 申请进度追踪
提供求职申请全生命周期的状态管理、时间线追踪和智能提醒功能。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Main Content Area

## Top Navigation Bar
- CareerMatch AI logo
- 主导航菜单：仪表盘、岗位管理、简历管理、个人档案
- 全局搜索框
- 通知铃铛图标
- 用户头像和下拉菜单

## Page Header Section
- 页面标题："申请追踪"
- 整体申请统计：
  - 总申请数量
  - 申请成功率（面试邀请率）
  - 平均响应时间
  - 本月新申请数量
- 状态分布图表：
  - 环形图显示各状态占比
  - 状态图例（草稿、已投递、面试中、已录取、已拒绝、已撤回）

## Status Filter Tabs
- 状态选项卡切换：
  - 全部申请（显示总数）
  - 草稿（待完善的申请）
  - 已投递（等待回复）
  - 面试中（进行中的面试流程）
  - 已录取（收到offer）
  - 已拒绝（未通过）
  - 已撤回（主动放弃）
- 快速筛选选项：
  - 时间范围（本周/本月/近3个月）
  - 岗位类型筛选
  - 公司筛选
  - 匹配度范围

## Application Cards List
- 申请卡片（每个岗位一张）：
  - 岗位信息区域：
    - 职位标题和公司名称
    - 工作地点和薪资范围
    - 申请日期和岗位来源
  - 状态指示器：
    - 当前状态标签（带颜色编码）
    - 状态持续时间（如"已投递 5天"）
    - 下一步行动提示
  - 匹配度显示：
    - 匹配百分比和星级
    - 匹配亮点简述
  - 进度时间线：
    - 水平时间轴显示关键节点
    - 当前位置高亮标记
    - 预期下一步时间
  - 快速操作按钮：
    - 更新状态
    - 查看详情
    - 添加备注
    - 设置提醒

## Detailed Timeline View
- 时间线详情（点击卡片展开）：
  - 垂直时间轴布局
  - 每个节点显示：
    - 具体日期时间
    - 事件类型（申请提交、简历查看、面试邀请等）
    - 详细描述
    - 附件文档（面试邀请邮件等）
  - 手动添加事件功能：
    - 电话沟通记录
    - 邮件往来
    - 面试反馈
    - 薪资谈判记录

## Interview Management
- 面试安排卡片：
  - 面试时间和地点
  - 面试类型（电话/视频/现场）
  - 面试官信息
  - 面试轮次（一面/二面/终面）
- 面试准备检查列表：
  - 研究公司背景
  - 准备STAR案例
  - 复习技术知识
  - 准备提问问题
- 面试倒计时提醒
- 面试结果记录：
  - 面试评价（自我感觉）
  - 面试反馈记录
  - 后续跟进计划

## Smart Reminders & Notifications
- 智能提醒设置：
  - 申请截止日期提醒（提前3天/1天）
  - 面试时间提醒（提前1天/2小时）
  - 跟进提醒（申请后2周无回复）
  - 感谢邮件提醒（面试后24小时）
- 提醒方式选择：
  - 邮件通知
  - 应用内推送
  - 手机短信（可选）
  - 日历集成

## Analytics Dashboard
- 申请效果分析：
  - 申请响应率趋势图
  - 不同岗位类型成功率对比
  - 简历版本效果对比
  - 申请时间段分析（最佳投递时间）
- 改进建议：
  - 基于数据的申请策略建议
  - 简历优化方向
  - 面试技能提升重点

## Bulk Operations
- 批量状态更新
- 批量添加提醒
- 批量导出申请记录
- 批量删除过期申请

## Export & Reporting
- 申请记录导出：
  - Excel格式详细报告
  - PDF格式申请总结
  - CSV格式数据文件
- 自定义报告生成：
  - 选择时间范围
  - 选择包含字段
  - 添加个人备注
