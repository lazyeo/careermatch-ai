# 求职仪表盘
为求职者提供统一的求职管理中心，展示求职进度、智能推荐和关键统计数据。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Main Content Area

## Top Navigation Bar
- CareerMatch AI logo
- 主导航菜单：仪表盘、岗位管理、简历管理、个人档案
- 全局搜索框（岗位快速搜索）
- 通知铃铛图标（新岗位推荐、申请状态更新提醒）
- 用户头像和下拉菜单

## Welcome Section
- 个性化欢迎语：显示用户姓名和当前求职状态
- 求职目标显示：目标岗位类型和地区偏好
- 当日求职建议：基于AI分析的每日求职任务提醒

## Statistics Overview
- 求职数据卡片：
  - 收集岗位总数（带本周新增标识）
  - 已申请岗位数量（带成功率显示）
  - 面试邀请数量（带待处理状态）
  - 匹配度>80%岗位数量（高价值机会）
- 求职进度环形图：显示申请状态分布
- 本周活动时间线：最近的申请和面试活动

## Smart Recommendations
- 智能推荐标题：今日为您推荐的岗位机会
- 推荐岗位卡片列表：
  - 岗位名称和公司信息
  - 匹配度评分（0-100分）和星级显示
  - 薪资范围和工作地点
  - 发布时间和申请截止日期
  - 一键查看详情和快速申请按钮
- 查看全部推荐链接

## Recent Activities
- 活动标题：最近的求职动态
- 活动时间线：
  - 新收集的岗位（浏览器扩展或手动添加）
  - 完成的匹配分析和生成的定制简历
  - 提交的申请和收到的面试邀请
  - 面试安排和结果更新
- 查看全部活动历史链接

## Quick Actions
- 快速操作按钮组：
  - 添加新岗位（手动输入URL或岗位信息）
  - 上传简历（创建新的简历版本）
  - 设置求职偏好（更新目标岗位类型）
  - 查看面试安排（跳转到日历视图）
