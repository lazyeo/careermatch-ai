# 岗位详情
提供单个岗位的全面信息展示、AI匹配分析和相关操作，采用三标签页布局提供不同维度的信息视图。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Main Content Area

## Top Navigation Bar
- CareerMatch AI logo
- 主导航菜单：仪表盘、岗位管理、简历管理、个人档案
- 全局搜索框
- 通知铃铛图标
- 用户头像和下拉菜单

## Job Header Section
- 返回岗位管理按钮
- 岗位基本信息：
  - 岗位标题和公司名称
  - 公司logo和行业标签
  - 工作地点和工作类型（全职/远程/混合）
  - 薪资范围和股权信息
  - 发布时间和申请截止日期
- 匹配度概览：
  - 大号匹配度评分（0-100分）
  - 五星评级显示
  - 匹配度等级标识（优秀/良好/一般/待提升）
- 申请状态标签和快速操作：
  - 当前申请状态（彩色标签）
  - 更改状态下拉菜单
  - 生成定制简历按钮
  - 开始申请按钮
  - 添加到收藏按钮

## Tab Navigation
- 岗位信息标签页（默认激活）
- 匹配分析标签页
- 生成文档标签页

## Job Information Tab
- 岗位描述部分：
  - 职位概述和主要职责
  - 技能要求和资格条件
  - 工作经验要求
  - 教育背景要求
- 公司信息部分：
  - 公司介绍和发展阶段
  - 公司规模和员工数量
  - 公司文化和价值观
  - 福利待遇详情
- 申请信息部分：
  - 申请方式和流程说明
  - 联系人信息
  - 附加材料要求
  - 面试流程预期

## Match Analysis Tab
- 综合匹配度分析：
  - 匹配度雷达图（9个维度）
  - 各维度得分详情和说明
- 优势分析部分：
  - 个人技能与岗位需求的匹配亮点
  - 经验优势和竞争力分析
  - 教育背景匹配度评估
- 差距分析部分：
  - 技能差距识别和改进建议
  - 经验不足领域和补强方案
  - 学习资源推荐
- SWOT分析矩阵：
  - Strengths（优势）：个人强项对比岗位需求
  - Weaknesses（劣势）：需要改进的技能领域
  - Opportunities（机会）：岗位发展前景分析
  - Threats（威胁）：竞争环境和挑战评估
- 申请建议部分：
  - 简历优化建议
  - 求职信写作要点
  - 面试准备重点

## Generated Documents Tab
- 定制简历部分：
  - 已生成的简历版本列表
  - 简历预览和下载功能
  - 重新生成简历按钮
  - 简历匹配度评分
- 求职信部分：
  - 已生成的求职信版本
  - 求职信预览和编辑功能
  - 重新生成求职信按钮
  - 个性化程度评估
- 面试准备材料：
  - 预测面试问题列表
  - STAR方法回答建议
  - 公司背景研究要点
  - 问题反问建议
- 申请历史记录：
  - 申请提交时间和方式
  - 申请状态变更历史
  - 沟通记录和反馈
  - 面试安排和结果
